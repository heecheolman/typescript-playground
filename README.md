# Typescript Playground
타입스크립트를 자유롭게 테스트하려고 만든 저장소입니다.

## Usage
```
$ git clone https://github.com/heecheolman/typescript-playground.git
$ cd typescript-playground
$ npm install
$ npm start 
```
